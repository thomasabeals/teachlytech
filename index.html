LinkedIn Web Scraper Project

Description: This s a web scraper to extract LinkedIn job postings and visualize them with a heat map by location.
  • Role: Developer and data analyst.
  • Goal: To automate job data collection and provide geographic insights on job availability.
  • Problem Statement: One of the biggest challenges to advancing in a career or starting a new one is locating an area that has the types of jobs you are looking for. Job seekers often lack data-driven insights on job distribution by location. This tool provides a visual overview, helping users target high-demand areas.
  • Process Approach: Developed a scraper with Python and Playwright to extract job titles, companies, and locations from LinkedIn URLs.
  • Tools: Used Pandas for data processing, Folium for heat map creation, and Playwright for scraping.
  • Solution: The scraper gathers job data, which is cleaned and fed into a geographic heat map to visualize job concentrations by city or region.
  • Results and Impact: Provides an interactive heat map, allowing users to quickly identify top job locations. Improved job search efficiency by highlighting regional job trends. This will be improved upon with more visualizaions empowering users to find new roles.
  • Reflection / Lessons Learned: Enhanced skills in web scraping and geospatial data visualization. Future improvements include refining location accuracy and integrating additional job details, such as industry and required skills.

Cryptocurrency Visualization Project for Potential Investors

Description Built a crypto exchange platform using CoinMarketCap API to display cryptocurrency data and help investors understand market trends.
  • Role:Full-stack developer, data analyst.
  • Goal: To create an intuitive interface for analyzing crypto market data for informed investment decisions.
  • Problem Statement: Cryptocurrency investors often lack easy-to-understand data visualizations. This project aims to simplify crypto market data by offering actionable insights and trend analysis in real time.
  • Process Approach: Integrated the CoinMarketCap API to retrieve live data on popular cryptocurrencies, then used data analysis to create insights into market trends. Tools: Python, CoinMarketCap API, Pandas for data manipulation, and Plotly/D3.js for interactive visualizations.
  • Solution: Created an interface with data visualizations, to highlight top-performing cryptocurrencies. Interactive filters allow users to view data by timeframe.
  • Results and Impact: The platform helps users quickly assess market health, identify trending assets, and compare cryptos effectively. Improved decision-making for investors through easy access to real-time and historical data.
  • Lessons Learned: Enhanced knowledge in API integration and financial data visualization. Future improvements include adding predictive analytics, portfolio tracking, and integrating advanced financial metrics.
